 Hotel Registration System (Java)
 Project Description

The Hotel Registration System is a console-based Java application designed to manage basic hotel operations such as guest check-in, room allocation, viewing room details, and guest check-out.
This project follows a menu-driven approach using while(true) and switch-case, making it simple and interactive.

 Objectives

To understand Java OOP concepts

To implement menu-driven programs

To manage hotel guest records efficiently

To use ArrayList for storing dynamic data

To practice real-life system design logic

 Technologies Used

Language: Java

JDK Version: Java 8 or above

IDE: VS Code 

Type: Console-based Application
